## Base Theme Features

- Home Page
- Blog
- Projects
- About
- Responsive layout with option for narrow/wide
- Tagging
- Animated header
- Social links

## Added customizations
- Custom Permalinks for blog posts and projects
- Minimal header for pages other than Home
- Hidden drafts
- Individual tag pages
